# Take You Online

Flagship-static education hub for programming depth, GATE CSE arcs, cryptography, and calibrated practice arenas.

## What's inside

| Path | Purpose |
|------|---------|
| `index.html` | Scroll-driven home (courses, GATE atlas, practice grid, FAQ) |
| `css/main.css` | Design tokens, layout, typography (Syne + DM Sans via Google Fonts) |
| `js/main.js` | Nav drawer, accordion, intersection reveals |
| `*_MCQ.html`, `dsa.html`, `coa.html`, `ga.html` | Subject hubs |
| `Python_blog.html`, `CPP_blog.html` | Long-form notes + unified site overlay |
| `HTML_blog.html`, `CSS_blog.html`, `Cryptography_blog.html` | Revamped prose tracks |
| `C_Language.html` | Interactive C MCQs |

No build step: open `index.html` locally or serve the folder with any static host.

## Lightweight stack

Semantic HTML · modern CSS (`clamp`, grid, `@media (prefers-reduced-motion)`). Content is deliberately static-first for speed and portability.
