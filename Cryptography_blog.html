Integrity · secrecy · proofs

Cryptography · contracts over adversarial wires

Every login, CDN fetch, webhook, and WASM bundle inherits decades of number theory distilled into keystreams and signatures. Learn primitives cleanly—protocols amplify or betray them instantly.

Goals, plainly

  • Confidentiality: ciphertext reveals nothing substantive without secrets.
  • Integrity: alterations trip MAC/AEAD tags before payloads mislead.
  • Authenticity: signatures + freshness (nonces, counters) tame impersonation and replay.

Symmetric reality

AES-GCM, ChaCha20-Poly1305, classic stream modes—bulk protection once endpoints synchronize keys. Operational discipline: randomized IVs/nonces unique per invocation; duplication collapses keystream secrecy.

Public-key choreography

RSA-era factoring hardness, ECC density and curve choices, ECDHE handshakes, hybrid post-quantum experiments—key distribution without couriers handcuffed to briefcases. Pedagogically Caesar ciphers visualize frequency attacks and motivate Kerckhoffs’ maxim: adversaries know algorithms; secrecy lives strictly in keys.

# toy Caesar shift — pedagogy only, never security
def rotate(text: str, k: int) -> str:
    alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    shifted = alphabet[k:] + alphabet[:k]
    trans = str.maketrans(
        alphabet + alphabet.lower(),
        shifted + shifted.lower(),
    )
    return text.translate(trans)

Cryptanalysis mindset

Treat ciphertext as hypotheses to disprove—linear/differential probes, birthday collisions on hashes, algebraic structure abuse. GATE framing often contrasts asymptotic hardness with pragmatic bit-security timelines.

Encryption without authenticated associated data leaks integrity; TLS record layers marry both—copy that instinct into every bespoke JSON token scheme you sketch.

Hashes, MACs, and bit budgets

Cryptographic hashes shrink messages into fixed-length digests for integrity checks and Merkle trees—yet MD5/SHA-1 are historical footnotes beside SHA-256/BLAKE-class designs. Birthday collision search for an n-bit hash needs on the order of 2n/2 trials—not 2n—so doubling bit width buys more than double the safety margin.

  • MAC / HMAC: keyed tags detect tampering inside closed systems sharing secrets.
  • AEAD suites: marry encryption + authenticity (GCM tags, Poly1305) so attackers cannot splice ciphertext without alarms.

Passwords need slow KDFs—not “encryption”

Human-selected passwords carry thin entropy compared to RNG keys—defenses stack unique salts plus memory-hard Argon2 (or calibrated scrypt/bcrypt workloads). Reversible password “encryption” usually fails once the database and key material stroll together offline.

Signatures, certificates, freshness

Asymmetric signing keys anchor trust for software releases and HTTPS: verification chains prove only as strong as root stores, revocation plumbing, and pinning policies allow. Pair signatures with freshness signals (nonces, anti-replay windows) lest captured payloads replay unnoticed.

Entropy discipline

Platform CPRNG APIs (getrandom, BCrypt APIs on Windows, Python’s secrets) beat ad-harvested mouse wiggles unless you consciously model entropy pooling. Statistical health checks matter on embedded MCU labs where RNG gates fail softly.

Threat sketches every answer should carry

Name actors (global passive adversary, rogue Wi-Fi café, insiders with packet capture), enumerate assets (session cookies, LMS grades), cite channels (HTTPS, QUIC WebTransport). Cryptography aligns with governance—perfect TLS cannot stop phishing that clones your OTP capture page.

Invention rule: before shipping “lightweight JWT encryption,” sanity-check AEAD framing, issuer audience claims, revocation, rotation, and downgrade stories—novel token schemes regress to TLS-era mistakes fast.

Protocol sociology

TLS authenticates endpoints via PKIX certificate graphs, binds negotiated suites against downgrade ladders, mixes long-lived authenticity with ephemeral forward secrecy wherever ECDHE participates. Messaging ratchets (Signal-style) chain forward secrecy to social onboarding rituals. Humans still revoke, pin, mistrust intermediates, argue over transparency logs—engineering security stitches math into operations calendars.

Quantum and curriculum hooks
