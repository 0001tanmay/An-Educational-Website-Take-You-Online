Integrity · secrecy · proofs

Cryptography · contracts over adversarial wires

Every login, CDN fetch, webhook, and WASM bundle inherits decades of number theory distilled into keystreams and signatures. Learn primitives cleanly—protocols amplify or betray them instantly.

Goals, plainly

  • Confidentiality: ciphertext reveals nothing substantive without secrets.
  • Integrity: alterations trip MAC/AEAD tags before payloads mislead.
  • Authenticity: signatures + freshness (nonces, counters) tame impersonation and replay.

Symmetric reality

AES-GCM, ChaCha20-Poly1305, classic stream modes—bulk protection once endpoints synchronize keys. Operational discipline: randomized IVs/nonces unique per invocation; duplication collapses keystream secrecy.

Public-key choreography

RSA eras, ECC density, ECDHE handshakes, hybrid post-quantum experiments—distribution without couriers handcuffed to briefcases. Pedagogically Caesar ciphers visualize frequency attacks and motivate Kerckhoffs’ maxim: adversaries know algorithms; secrecy lives strictly in keys.

# toy Caesar shift — pedagogy only, never security
def rotate(text: str, k: int) -> str:
    alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    shifted = alphabet[k:] + alphabet[:k]
    trans = str.maketrans(
        alphabet + alphabet.lower(),
        shifted + shifted.lower(),
    )
    return text.translate(trans)

Cryptanalysis mindset

Treat ciphertext as hypotheses to disprove—linear/differential probes, birthday collisions on hashes, algebraic structure abuse. GATE framing often contrasts asymptotic hardness with pragmatic bit-security timelines.

Encryption without authenticated associated data leaks integrity; TLS record layers marry both—copy that instinct into every bespoke JSON token scheme you sketch.

Protocol sociology

TLS verifies certificate graphs resists downgrade; messenger ratchets blend forward secrecy with social trust bootstrapping. Engineering security is choreography between math and human operations—revocation, pinning, transparency logs.
